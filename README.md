# Azerbaijan-International-Trade-Analysis
The data in this analysis is based on the **State Statistical Committee of the Republic of Azerbaijan**. 
Within this data, you can examine annual **import and export trends,**
**Trading partner countries**, 
**Statistics on key commodities**,
*Tthe value of imports and exports** by country.

85% of trade data of Azerbaijan Republic was cleaned in **Python-pandas** (Left ones were in **Power Query**).
Full analyzing was prepared in **Power BI**
